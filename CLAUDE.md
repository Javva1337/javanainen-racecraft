## OBLIGATORISKT när text skrivs eller ändras — VIKTIGT

När jag skriver, skriver om eller putsar **användarvänd text** i det här projektet
(copy på sidor, hero/rubriker, CTA:er, bildtexter, rapporttext, meta-taggar, e-post,
knapptext m.m.) ska följande skills ALLTID köras, i den här ordningen:

1. **`copywriting`** — för att skriva eller förbättra själva copyn.
2. **`seo-audit`** — optimering för **Google** (teknisk/on-page, meta, rubrikstruktur).
3. **`ai-seo`** — optimering för **AI-sök** (bli citerad av ChatGPT/Perplexity/AI Overviews; AEO/GEO).
4. **`humanizer`** — ALLTID som sista pass, för att ta bort AI-drag och landa Rickards ton.

Detta gäller även små textändringar. Hoppa inte över stegen för att texten "är kort".

### Ton (kopplat till minnet)
Rickard vill aldrig låta kaxig — ödmjuk ton. `humanizer` + `copy-editing` vid
omskrivningar. Se minnesfilen `ton-odmjuk-copy.md`.

### Undantag
Ren kod, kommentarer, commit-meddelanden, config och intern dokumentation räknas
INTE som användarvänd text och kräver inte skill-kedjan ovan.
